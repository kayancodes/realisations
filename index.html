Digiplanet : hackathon

Ux Ui Design
 

  • Rôle joué

    Ux Ui Designer

  • Outils

    Méthodologie de design UX - Marvel app

  • Nombre de pages

    8

  • Client

    Communauté d'agglomération Roissy Pays de France

  • Type de project

    Challenge hackathon

  • Date de réalisation :

    2019

  • Contexte:

    Maquette interactive du site mobile réalisé lors du hackathon Digiplanet.

  • Consignes :

    "Réalisation du design d'une application qui permettra aux utilisateurs de suivre leur impact numérique (taux d'émission carbone) et qui les aidera à adopter les gestes écologiques."

  • Fonctionalités attendues :

    - Affichage de l’empreinte carbone
    - Affichage des gestes écoresponsables
    - Partage sur les réseaux sociaux (Facebook et Twitter)
    - Fermeture des pages inactives
    - Tri des mails

  • Remarque :

    En qualité de capitaine d'équipe j'ai eu l'opportunité de présenter le projet devant un jury composé notamment de :
    - M. SOUFIR, Vice-président en charge du développment numérique, Communauté d'agglomération Roissy Pays de France.
    - Mme. BOUFFARD, Directrice de l'économie des territoires, de l'innovation et du numérique, Communauté d'agglomeration Roissy Pays de France.
    - M. ROUASS, Responsable promotion, Simplon.
    - M. TOISSON, Scrum Master, Capgemini

Facil'Hote

Ux Ui Design
 

  • Rôle joué

    Ux Ui Designer

  • Outils

    Méthodologie de design UX - Photoshop - Marvel app

  • Nombre de pages

    7

  • Client

    Hôtel Royal Montreuil

  • Type de projet

    Projet d'étude

  • Date de réalisation :

    2019

  • Contexte:

    Projet de groupe réalisé en collabaration avec le gérant de l'hotel Royal Montreuil, dans le cadre du passage du titre professionnel Concepteur et Développeur d'Applications, de 2019 à 2020.

  • Consignes :

    "Réalisation du design d’une application web qui aura pour but de remplacer le système de gestion « papier » actuel de l’hôtel Royal Montreuil. L’application étant destiné à divers utilisateurs, il devra être clair, facilement abordable, accessible et compréhensible par tous."

  • Fonctionnalités attendues :

    L’application devra répondre aux objectifs définis par le client :
    - Objectif 1 : Automatisation du planning de réservation
    - Objectif 2 : Automatisation du système de Facturation
    - Objectif 3 : Gestion de l'état de propreté des chambres
